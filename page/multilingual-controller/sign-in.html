Sign in

1. UI

Open http://localhost:3000 to open Multilingual Controller.
Some demo account:

- Role: username/password
- Admin: 1/12345789
- Two_role: 2/123456789
- Developer_role: 3/123456789
- Translator_role: 4/123456789

2. Sign in failed

A message on the top notification sign in failed.

3. Sign in with Developer role

Go to Developer Module.

4. Sign in with Translator role

Go to Translator Module.

5. Sign in with dual role

Default go to Developer Module. User can choose other module by choose the option box in the header.

6 .Sign in with Admin role

Default go to Compiler Module. User can choose other module by choose the option box in the header.
