Compiler Module

To automatically convert a component with a source language to a target language. The website be loaded fast as “static” web-pages with standardized translation monitored by human.

User benefits:

  • Fast loading.
  • Consistent texts.
  • Monitoring and comprehensive translations.

1. UI

Employee has role: admin can access to this module.

2. Load projects

Same load projects in Developer Module.

3. Load components

Same load components in Developer Module.

4. Auto check

This process is configured by two sub-processes.

  • Checking for non-translated texts.
  • Checking for missing markers.

Admin must upload the source code to folder /tmp/adminId.projectId.componentId.langId in the server before running Auto check and Compile!

If not exist folder:

Passed checking for non-translated texts and missing markers.

Not passed checking for non-translated texts but passed missing markers.

Passed checking for non-translated texts but not passed missing markers.

Not passed checking for non-translated texts and not passed missing markers.

5. Compile

To automatically convert a component with a source language to a target language.

Output: a new folder with name path + "_translated", can download by click the link in the right box.

Note: can compile only when auto check is completed !

If it not passed auto check, a message box will be show to confirm force compile.
