How to run

1. Requirement

  • Node.js: https://nodejs.org - Version 7.0.0 or later
  • Neo4j database: https://neo4j.com/ - (Optional)
  • Redis database: https://redis.io/ - (Optional)
  • SQLite database: https://www.sqlite.org - Version 3.0.0 or later (Optional)
  • MySQL database: https://www.mysql.com - (Optional)
  • CouchDB database: http://couchdb.apache.org - Version 2.0.0 or later (Optional)

2. Install

$ git clone https://gitlab.com/listingexpress/LE-components.git

3. Setup dbAPI-server

$ cd LE-components/dbAPI-server
$ npm install
To know how to use dbAPI-server go to section dbAPI in left menu.

4. Run dbAPI-client

$ cd LE-components/dbAPI-client
$ npm install
$ npm start

Now, server start on port 4000.
To know how to use dbAPI-client go to section dbAPI in left menu.
