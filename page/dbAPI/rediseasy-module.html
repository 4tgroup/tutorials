RedisEasy Module

To wrap the set/add, get, del Redis command to one input object parameter. This module executes the command via ioredis

1. Usage

Using in Node.js

var RedisWrapper = require('./dbAPI-server').RedisWrapper;
                         
RedisWrapper.set(input, function (output) {   //Can replace .set by .get or .del
    //Handle output
});

add command will execute if the variable of set command do not exist in database.

2. Input

The JSON object with the format:

params = {
        "variable": "name of variable" or [list of variable names],
        "value": "Strings datatype value" or [] or {},
        "index": [],
        "other": []
        }

Parameters meaning:

  • variable (key): contain single string variable or array variable.
  • value: contain single string value or array values or object {hkey:hvalue} (Hashes datatype).
  • index: contain the index of command
      set command:
    • [0]: insert values to the head of the list.
    • [-1]: insert values to the tail of the list.
    • ["pivot"]: insert value to BEFORE|AFTER (default AFTER) pivot value.
    • [10, 20, 30]: define score for corresponding values in value.
      • get command:
    • [index]: get a value in list with an index position.
    • [startidx, stopidx]: get values in list/sorted set with an index range.
    • ["hkey1","hkey2","hkey3"]: get a values in hash with index keys.
      • del command:
    • [0]: remove the 1st element.
    • [-1]: remove the last element.
    • [index]:
      • index > 0 - Remove elements equal to value moving from head to tail.
      • index < 0 - Remove elements equal to value moving from tail to head.
      • index = 0 - Remove all elements equal to value.
    • [10, 20]: removes all elements in the sorted set stored at key with rank between startidx and stopidx
    • ["hkey1","hkey2","hkey3"]: removes the specified hkeys from the hash stored at variable (key).
  • other: contain the other information like command option.
    • ["EX", seconds]: expire time in seconds
    • ["PX", milliseconds]: expire time in milliseconds
    • ["BEFORE|AFTER"]: insert value BEFORE|AFTER a pivot to list.
    • ["WITHSCORES"]: get score of corresponding members in sorted set.
    • ...

3. Example parameters

Set/Add command:

  • Strings datatype

    • Set a variable "String":"Hello dbAPI" and will expire in 10s.

      params = {
        "variable": "String",
        "value": "Hello dbAPI",
        "index": [],
        "other": ["EX", 10]
    }    
//If variable "String" exists ("String":"Hello "), this command will be Add command and append "Hello dbAPI" to "String" ("String":"Hello Hello dbAPI")

    • Set multiple variable and values "name1":"val1", "name2":"val2", "name3":"val3"and will expire in 100 milliseconds.

      params = {
        "variable": ["name1", "name2", "name3"],
        "value": ["val1", "val2", "val3"],
        "index": [],
        "other": ["PX", 100]
    }
  • Lists datatype

    • Insert an array values to the head of a variable, also set expire time if you want.

      params = {
        "variable": "list_name",
        "value": ["val1", "val2"],
        "index": [0],
        "other": ["EX", 10]
    }

    • Insert a value before or after a pivot of an existing variable. You also define expire time at the second position in array

      params = {
        "variable": "exist_list_name",
        "value": ["value"], //Array has only one value
        "index": ["pivot"],
        "other": ["BEFORE", ["EX", 10]] //LINSERT option (default AFTER)
    }
  • Sets datatype

    • Insert an array unique values to a variable.

      params = {
        "variable": "set_name",
        "value": ["val1", "val2"],
        "index": [],
        "other": []
    }
  • Sorted sets datatype

    • Insert an array unique values with score to a variable.

      params = {
        "variable": "sorted_set_name",
        "value": ["val1", "val2", "val3"],
        "index": [10, 20, 30],  //Score of the values
        "other": []
    }
  • Hashes datatype

    • Insert an hash object {"hkey1":"hval1", ...} to a variable.

      params = {
        "variable": "hash_name",
        "value": {"hkey1":"val1", "hkey2":"val2", "hkey3":"val3"},
        "index": [],
        "other": []
    }

Get command:

  • Strings datatype

    • Get a value of a variable.

      params = {
            "variable": "String", //for get multiple use array ["name1", "name2", "name3"]
            "value": [],
            "index": [],
            "other": [] 
        }
  • Lists datatype

    • Get array values of a variable in a range from position 0 to 10.

      params = {
        "variable": "list_name",
        "value": [],
        "index": [0, 10],
        "other": [] 
    }
  • Sets datatype

    • Get all array unique values of a variable

      params = {
        "variable": "sets_name",
        "value": [],
        "index": [],
        "other": [] 
    }
  • Sets datatype

    • Get values from hash object with corresponding hash keys.

      params = {
        "variable": "hash_name",
        "value": [],
        "index": ["hkey1","hkey2","hkey3"],
        "other": [] 
    }

Del command:

  • Strings datatype

    • To delete string varaiable, we set this variable expire in 0 ms (Redis does not support command to remove Strings varaiable)

      params = {
        "variable": "name",
        "value": [],
        "index": [],
        "other": ["PX", 0] //Expire in 0 millisecond
    }
  • Lists datatype

    • Remove 2 values with value is "ABC" moving from tail to head.

      params = {
        "variable": "list_name",
        "value": ["ABC"], //if value exist, del command will depend on this value.
        "index": [-2], //see more [index] in option
        "other": []
    }
  • Sets datatype

    • Remove "val1" and "val2" in the unique "set_name" array

      params = {
        "variable": "set_name",
        "value": ["val1", "val2"],
        "index": [],
        "other": []
    }
  • Hashes datatype

    • Remove specified hash keys from hash object.

      params = {
        "variable": "sorted_set_name",
        "value": [],
        "index": ["hkey1","hkey2","hkey3"],
        "other": []
    }
