SQLParser Module

To parse the SQL query to the AST (Abstract Syntax Tree) JSON.

1. Usage

Using in Node.js

var SQLParser = require('./dbAPI-server').SQLParser;

var SQLQuery = "SELECT * FROM users";                            
SQLParser(SQLQuery, function (err, ast) {
    //Handle err & ast result
});

2. Input

The SQL query. Support CREATE, INSERT, UPDATE, SELECT, DELETE, ALTER, INDEX query. Some simple SQL queries example:

---------CREATE---------- 
CREATE TABLE users (id int, name varchar(50));
---------INSERT---------- 
INSERT INTO users VALUES (1, "John"), (2, "Diana");
---------UPDATE---------- 
UPDATE users SET name = "Laura" WHERE id = 2;
---------SELECT---------- 
SELECT * FROM users WHERE name = "John";
---------DELETE---------- 
DELETE FROM users WHERE name = "Laura";
---------ALTER---------- 
ALTER TABLE users ADD new_column TEXT;
---------INDEX---------- 
CREATE INDEX idx_name ON users(name);

The SQL queries can be more complicated.

3. Output

The AST JSON. The AST JSON look like: 

{
  "type": "statement",
  "variant": "list",
  "statement": [
    {
      "type": "statement",
      "name": {
        "type": "identifier",
        "variant": "table",
        "name": "users"
      },
      "variant": "create",
      "format": "table",
      "definition": [
        {
          "type": "definition",
          "variant": "column",
          "name": "id",
          "definition": [],
          "datatype": {
            "type": "datatype",
            "variant": "int",
            "affinity": "integer"
          }
        }
      ]
    }

The error response: 

{
  "status":"error_code",
  "message":"error_message/warning_message"
}
Status error response:
  • 0 - success.
  • 199 - warning.
  • 400 - client error.
  • 500 - server error.

          "type": "expression",
          "variant": "list",
          "expression": [

              "type": "literal",
              "variant": "decimal",
              "value": "1"

            {

            }
          ]
        },
        {
          "type": "expression",
          "variant": "list",

              "type": "identifier",

          ]

      ]
    }
  ]
}
