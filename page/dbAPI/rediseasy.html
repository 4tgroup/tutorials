RedisEasy

Provide set, add, get, and del command for easily interating with Redis database using the same template format. This module executes the command via 'ioredis' package

1. Usage

Using in Node.js

var Redis = require('./dbAPI-client').RedisEasy;
var redis = new Redis(); // connect to 127.0.0.1:6379, db: 0
// var redis = new Redis({"host":"192.168.1.1", "port":"6380", "db": 1});

// set, add, del
redis.set(data); //Can replace .set by .add or .del

// get
var rs = redis.get(data, function (output) {   
    if (ouput.status == CONST.STATUS_OK){
        return output.results;
    } else {
        return ({"Msg":output.message});
    }
});

add command will execute if the variable of set command do not exist in database and vice versa.

2. Examples

2.1. Set command:

  • String datatype

    • Create variable "var1" with the string value of "Hello" and expiration time of 10 seconds.

      var data = {
        "variable": "var1",
        "value": "Hello",
        "other": ["EX", 10]
        };

redis.set(data);

  • Hash datatype

    • Insert the hash object {"hkey1":"hval1", ...} to a new variable hash1.

      
redis.set({"variable": "hash1", "value": {"hkey1":"val1", "hkey2":"val2", "hkey3":"val3"}});

2.2. Add command:

  • List datatype

    • Insert the (repeating and unsorted) array ["lval1", "lval2", "lval3", "lval4"] to the head of an existed variable list1.

      var data = {
        "variable": "list1",
        "value": ["lval1", "lval2", "lval3", "lval4"],
        "index": [0],
        };

redis.add(data);

  • Set datatype

    • Insert the (none-repeating and unsorted) array ["sval1", "sval2", "sval3"] to the variable set1.

      var data = {
        "variable": "set1",
        "value": ["sval1", "sval2", "sval3"]
        };

redis.add(data);

2.3. Get command:

  • String datatype

    • Get the value of the variable var1.

      
var var1 = redis.get({"variable": "var1"});
  • List datatype

    • Get values of the (repeating & unsorted) array from the variable list1 with the position range from 1 to 3.

      
var list1 = redis.get({"variable": "list1", "index": [1, 3]});

2.4. Del command:

  • Set datatype

    • Delete the entire set1 variable

      
redis.del({"variable": "set1"});

  • Hash datatype

    • Delete the values with hash keys "hkey1" and "hkey2" from hash object hash1.

      var data = {
        "variable": "hash1",
        "index": ["hkey1", "hkey2"]
        };

redis.del(data);
