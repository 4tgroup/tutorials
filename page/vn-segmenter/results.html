Results

1. Dataset

To experiments, we collect dataset from the VLSP project .
It is a collection of many articles from the "Politics – Society" section of the Vietnamese newspaper "Tuổi trẻ" (The Youth).
All words that have been manually spell-checked and segmented by linguists from the Vietnam Lexicography Center.

There are 6 datasets:

  • The first dataset includes 7K sentences (about 2M syllables), each sentence is each line, has not HTML tags.
  • The second dataset include 27K sentences, same the first dataset but still has HTML tags.
  • The third dataset include 70K sentences, same the second dataset.
  • The fourth, fifth, sixth dataset include 7K, 27K, 70K same the first, second,third dataset but without diacritics.

The features of dataset:

  • The syllables in a multisyllabic word connected by "_".
  • In the data files are still the HTML tags, empty line, special characters.
  • The input file has extension *.raw, output file has extension *.seg.
  • Lowercase words


2. Execution time


Our tool, vnSegmenter.js has the fastest execution time in all dataset (~100 times with vnTokenizer). vnTokenizerhas the longest execution time in dataset 7K and 70K. DongDu, UETsegmenter, JVnTextpro has average execution time.


3. Precision percentage


UETsegmenter obtains an up-to-date highest recision. Next is vnTokenizer. JVnTextpro has lowest precision. Our tool, vnSegmenter.js has average precision.


4. F1-score


UETsegmenter obtains an up-to-date highest F1. Next is vnTokenizer. JVnTextpro has lowest F1. Our tool, vnSegmenter.js has average F1.
